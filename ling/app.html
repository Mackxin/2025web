纯纯看番

使用 Compose 开发的 Android 看番软件，支持多番剧源

  • 纯纯看番源码完全免费，在 Github 开源
  • 系统的插件化框架，通过安装插件来获取更多功能
  • 自动管理你追的番剧，确保第一时间获得更新
  • 一直在用的软件

官网 开源地址

Kotatsu

一个简单方便的 Android 开源漫画阅读器

  • 可以在其中比以往任何时候都更容易地找到和阅读您最喜爱的漫画
  • 支持超过 1000+ 个来源
  • 快速轻松地搜索感兴趣的标题
  • 支持中文

官网 开源地址

Kazumi

使用 flutter 开发的基于自定义规则的番剧采集与在线观看程序

  • 使用最多五行基于 Xpath 语法的选择器构建自己的规则
  • 支持规则导入与规则分享
  • 支持基于 Anime4K 的实时超分辨率

Seal

一个简单的 Android 视频/音频下载器

  1. 从 yt-dlp 支持的视频平台下载视频和音频文件
  2. 遵循 Material Design 3 设计规范，动态色彩应用界面
  3. 从数千个网站下载视频与音频，以 Material You 设计风格呈现

Aves

一个图库和元数据资源管理器应用程序

  1. 它会扫描您的媒体收藏以识别动态照片、全景图（又名照片球体）、360° 视频以及 GeoTIFF 文件
  2. 可以处理各种图像和视频，包括典型的 JPEG 和 MP4，也可以处理更多奇特的东西
  3. 导航和搜索是 Aves 的重要组成部分

开源地址

CellularZ

一款电信小区、信号质量及WiFi信息、信道查看软件

  1. 双卡手机网络信息查看（SIM卡信息、服务小区信息、服务小区信号质量，部分手机可获取邻小区信息）
  2. WIFI信息（已连接信息、附近WIFI列表、WIFI信道2.5及5GHz）
  3. 当前位置信息，设备信息（电池信息，硬件信息、系统信息）
  4. 速度测试，测试轨迹，室内覆盖
